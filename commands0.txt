commands for setting up gcc and root software

CMS users:
Modern gcc and root version are setup when setting up a CMS release 7.0.0 or greater under SL5
log into the lpc sl5 machines, type:
ssh -X cmslpc-sl5.fnal.gov
type:
source /uscmst1/prod/sw/cms/cshrc prod
scram p CMSSW CMSSW_7_0_0
cd CMSSW_7_0_0
cmsenv
cd ..
git clone --branch production https://github.com/herndon/FNALComp.git FNALComp
make

Note that at each new login you have to run the "source /uscmst1/prod/sw/cms/cshrc prod" command
and cd to the CMSSW release and run "cmsenv" before cding to your git release to work.
